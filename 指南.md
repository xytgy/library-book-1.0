# 启动指南

## 首次启动

### 1. 环境要求

- Java 17+
- Maven 3.6+
- MySQL 5.7+

### 2. 初始化数据库

```bash
mysql -u root -p123456 < database/init.sql
```

### 3. 配置数据库密码

编辑 `backend/src/main/resources/application.properties`：

```properties
db.url=jdbc:mysql://localhost:3306/library_management?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=UTF-8&allowPublicKeyRetrieval=true
db.user=root
db.password=你的数据库密码
```

### 4. 编译打包

```bash
cd backend
mvn clean package -DskipTests
```

### 5. 运行

```bash
java -jar target/library-backend-1.0.0.jar
```

---

## 日常启动

```bash
cd backend
java -jar target/library-backend-1.0.0.jar
```

> 注意：如果修改了 Java 代码，需要先重新编译：
> ```bash
> mvn clean package -DskipTests
> ```

## 重置数据库

如果需要清空数据重新开始：

```bash
mysql -u root -p123456 < database/init.sql
```

## 常见问题

### 数据库连接失败
检查 `application.properties` 中的数据库用户名和密码是否正确。

### 中文乱码
确保终端支持 UTF-8 编码。Mac/Linux 终端默认支持，Windows 可能需要设置：
```bash
chcp 65001
```
