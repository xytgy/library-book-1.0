# 启动指南

## 首次启动

### 1. 配置数据库

编辑 `backend/src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/library_management
    username: root
    password: 123456
```

### 2. 初始化数据库

```bash
mysql -u root -p < database/init.sql
```

### 3. 安装前端依赖

```bash
cd frontend
npm install
```

### 4. 启动服务

终端1（后端）：
```bash
cd backend
mvn spring-boot:run
```

终端2（前端）：
```bash
cd frontend
npm run dev
```

---

## 日常启动

终端1：

## 关闭服务

### 方法1：在终端按 Ctrl+C（推荐）
- 在运行服务的终端窗口按 `Ctrl + C`

### 方法2：如果 Ctrl+C 无效，用命令行强制关闭

**检查服务是否运行：**
```bash
netstat -ano | findstr ":8080 :5173"
```

**停止后端（端口 8080）：**
```bash
# 找到 PID 后替换下面的 <PID>
taskkill /F /PID <PID>
```

**停止前端（端口 5173）：**
```bash
# 找到 PID 后替换下面的 <PID>
taskkill /F /PID <PID>
```

### 重置数据库（可选）

如果需要清空数据重新开始：
```bash
mysql -u root -p < database/init.sql
```
